# Calculator

## Requirements

Perform basic arithmetic operations.
Display results dynamically.

## Things to Use

TextInput for number entry.
Button for operations.
Flexbox for styling.

## Acceptance Criteria

Correct calculations for all operations.
Clear and intuitive UI.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
